# 📊 Quality_and_Defect_Reduction  

This project focuses on reducing defects using data-driven techniques. It uses Excel for data preparation, Python in Google Colab for analysis, and Power BI for visualization. The goal is to identify patterns, perform root cause analysis, and improve overall process quality.

---

## 🎯 Objectives  

- 🔍 Identify defect patterns and trends  
- 🧠 Perform root cause analysis  
- 📊 Monitor quality KPIs (Defect Rate, Yield)  
- ⚙️ Improve operational efficiency  

---

## 🛠️ Tools & Technologies  

- 📄 Excel – Data cleaning & preprocessing  
- 🐍 Python (Google Colab) – Data analysis & EDA  
- 📊 Power BI – Dashboard & visualization  
- 🗄️ MySQL – Data querying & insights  
- 🌐 GitHub – Version control
